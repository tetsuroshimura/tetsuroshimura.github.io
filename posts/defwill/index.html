Def Will Official Web Site

avexのガールズグループ Def WillのWebサイトを製作。
今っぽい女子感を目指しながら、構造的にも「ファンにきちんと見てもらえる」という部分を大事にして設計を行なった。そのためにサイトの根幹部分にSNSの更新を組み込み、毎回見るたびに新しい情報が見られるようにした。野心的なポイントとしては、CMSに頼らずに運用できる（既存のSNSプラットフォームを更新することでWebサイトに反映される）ものを目指した。このあたりの仕組みの一部にYahoo! JAPANの新しいサービス、myThingsを利用している。
デザインとしては、Apple MusicやInstagramなどの普段使っているアプリに寄せていくことで、アプリを横断してDef Willの情報を見ていくような感覚を再現。各デバイスへのレスポンシブデザインに関しても独自の回答ができたと思っている。その他、時間帯によってサイトの表示が切り切り替わるなどの細かい小ネタを詰め込んだ。
撮影とグラフィックは雑誌のNYLONを手がけるカエルム社が担当。
